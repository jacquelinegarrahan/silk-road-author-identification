UPDATED 8/8/2013 - 2 VERSIONS AVAILABLE

Since Freedom Hosting went down, which is one primary services (well, the provider of many primary services) I use for uploading images since Imgur.com no longer allows uploads over TOR, I decided to create my own. After fearing a DNS, IP, or MAC address leak I ditched my old server (which didn't have much hosted on it anyways, FH seemed like a secure platform). So, with new hardware* I'm slowly trying to provide services that are needed in the Silk Road and TOR community.


My first - Anonymous Image Uploader

100% Javascript Free
You should still disable Javascript when using TOR.
ImagePro V1

http://37otxza6ovc3nq3c.onion/app/imagepro/
This version is very fast but is limited to 1 file per upload session, no change in filename, a maximum final size of 1024x800 (images are resized), and a maximum size of 10240mb
ImagePro V2

http://37otxza6ovc3nq3c.onion/app/imageprov2/
This version is slower, but can handle up to 10 sequential file uploads, does not require resizing, renames your file for privacy and duplicate filename issues, and does not output a thumbnail.

If you reach a "website cannot be found" error, try going to this address first: http://37otxza6ovc3nq3c.onion:80



It's very minimalistic and doesn't have many bells and whistles (nearly none) which allows it to maintain decent speed in uploading even behind TOR over VPN. It does not alter your image or add any metadata, but in turn it DOES NOT AUTOMATICALLY REMOVE METADATA. The ImagePro page has a link to an EXIF Stripper I use so users can remove the EXIF data (if they have not already) before they upload.

Features & Specs:

    One photo per upload
    10mb Image Size Limit
    .bmp, .jpg, .jpeg, .png, .gif, and .tif
    Regular and thumbnail images are produced
    Regular Image size is 1024x800
    Thumbnail size is 256x192

I should note that files are uploaded to a public directory, meaning that users can explore all the photos that have been uploaded. For this reason, you should not include a filename that may be used to identify you. This issue has been fixed. You must now use the direct link, and will not be able to explore the directory.

Image Directories - These are now private, only a direct link will work.

    http://37otxza6ovc3nq3c.onion/app/imagepro/normal/
    http://37otxza6ovc3nq3c.onion/app/imagepro/thumbnail/

EXIP Data Remover (stripper.exe)

http://37otxza6ovc3nq3c.onion/app/stripper/stripper.exe

For now it is the first (and only) working application on my new server, which in light of the Freedom Hosting disaster I decided I should invest in new server hardware.

Currently I don't seem to be able to get the application working with onion.to through the clearweb, but due to security concerns I always recommend users to use TOR and never use a clearnet tunnel. On top of that, I also recommend every TOR user invest in a VPN service. You can find plenty that accept bitcoin and are anonymous. You can then edit Vidalia to run the VPN every time you start TOR.

I'll be getting the forums up next, and then since I have no alternative I'll host my webstore here as well. The advantage is I'll be able to integrate BTC Merchant API's directly into the page with the new server, as the old ones had severe limitations. Of course, I'll always offer SR escrow, although I am looking into alternate Escrow services (over .onion) for sales outside of SR.

This service should not be confused with any other service that may be called ImagePro.
