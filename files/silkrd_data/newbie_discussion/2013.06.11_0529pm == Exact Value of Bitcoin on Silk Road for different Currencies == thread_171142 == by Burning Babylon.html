As one can see from the custom listings at the start the value has fallen 8.0%, 8.0%, 8.5% and 8.2% respectively. One can gather the discrepancies with the last two listings is because Silk Road only displays two decimals, so I will only calculate on the first two for the sake of simplicity.

$102.71009 USD / 1.0 BTC = 664.73191 SEK versus $101.22501 USD / 665.04837 SEK = 1.446 % decrease / 0.048 % increase
$128.38761 USD / 1.25 BTC = 830.91486 SEK versus $126.53127 USD / 831.31050 SEK = 1.446 % decrease / 0.048 % increase

This basically means using http://bitcoinexchangerate.org will show the USD value of a Silk Road listing incorrectly by 1.446 %, and the SEK value incorrectly by 0.048 % - they simply don't synch with the same data. I'm not entirely sure how the mechanics work here with one currency deviating by more than one percent from another though.  :o

1 day / 15 minutes = Median Price: 102.5 / WMA: 100.7 versus 1 day / 15 minutes = Median Price: 110.5 / WMA: 109.1 | Median Price: 7.805 % increase / 8.342 % increase
1 day / Daily = Median Price: 100.6 / WMA: 118.4 versus 1 day / Daily = Median Price: 110.0 / WMA: 117.6 | Median Price: 9.344 % increase / 0.676 % decrease

Here one can conclude that by using the Chart Type: Median Price the most accurate listing is 1 day / 15 minutes, only deviating by 0,195 % and 0,342 % respectively. To summarize for anyone who didn't want to read or couldn't follow what I was writing, the most accurate method to be able to track a value on Silk Road is to use the following two links:

http://bitcoincharts.com/charts/mtgoxUSD#rg1zig15-minztgMza1gWMAzm1g96zm2g90 ( Chart Type: Median Price - 1 day / 15 minutes )
http://bitcoincharts.com/charts/mtgoxUSD#rg1zig15-minztgSza1gWMAzm1g96zm2g90 ( Chart Type: CandleStick - 1 day / 15 minutes )

Add the values of Median Price and WMA together on the first link, then divide by 2.
Add the values of Op, Hi, Lo, Cl and WMP together on the second link, then divide by 5.
